Web  |  Projects

sample-jam | GitHub

Using the Web Audio API and RecorderJS, sample-jam lets users build and play their own instrument. Once signed up, users can save and load their samples. Utilized MongoDB and ExpressJS for the back end, JQuery up front.

InvestWatch | GitHub

Investors can get up to the minute stock quotes and after signing up, can save stocks to a personal watch list. Built on a Mongo/Express server with Angular on the front end.

JeopPrep | GitHub

Play actual Jeopardy games from the past. Compete against other players for all time high scores. Built with ExpressJS, the app draws from an API serving thousands of real Jeopardy clues.

MyTravels | GitHub

Create a personal scrapbook of your travels from the past and future. This Ruby on Rails app uses a PostgreSql database to store trip information. Angular handles the front-end rendering.

MenuFlow | GitHub

Restaurant management and point of sale app built with Ruby on Rails. Allows for admin to manage menu and staff, and waitstaff to create parties, take orders and view receipts.

JSBoggle | GitHub

A recreation of the classic word game using front-end Javascript. This project is not currently hosted online, but is an example of what I was able to build after only a few weeks coding.
